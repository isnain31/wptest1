== Description ==

CleanBlog theme is designed for all kind of blogs such as fashion blog, lifestyle blog,  travel blog, food & recipe blog, photography blog,  simple magazine, diaries or any other blogging website. CleanBlog theme is clean, latest design, easy to customizable, SEO friendly, bootstrap-based fully responsive to any devices and it has powerful custom widgets and customizer theme options.There are many features- standard, Grid, and lists layouts, Left and right sidebar, custom sidebar widgets and footer widgets, customizer options, Woocommerce support and much more... Online demo: http://airthemes.net/cleanblog/demo/ Theme Features: https://airthemes.net/cleanblogg/ Documentation: https://airthemes.net/cleanblogg/documentation/ Support: https://airthemes.net/support/  

== Installation ==

1. Log into the admin panel, go to **Appearance -> Themes** and click the **Add New** button.
2. Search **cleanblog** and press the **Enter** key.
3. Click on the **Activate** button to use your new theme right away.

or

1. Upload the **cleanblog** folder to the **/wp-content/themes/** directory
2. Activate by going to **Appearance -> Themes**
