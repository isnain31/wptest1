== Description ==

CleanBlog theme is designed for all kind of blogs such as fashion blog, lifestyle blog,  travel blog, food & recipe blog, photography blog,  simple magazine, diaries or any other blogging website.CleanBlog theme is clean, latest design, easy to customizable, bootstrap based fully responsive for any devices  and it has powerful custom widgets and customizer theme options.There are many features- standard, Grid, and lists layouts,Left and right sidebar, custom sidebar widgets and footer widgets, and much more..

== Installation ==

1. Login to Airthemes.net account and download cleanblog-pro-v1.x.x.zip on download page. (Or you can download cleanblog-pro-v1.x.x.zip file from payment details page or from email receipt)
2. Login to your site admin page and go to  Appearance -> Themes.
3. Click on the  Add New button.
4. On the next screen, click on the Upload Theme button at the top.
5. Now, click the  Choose File button and choose the cleanblog-pro-v1.x.x.zip file that you downloaded earlier. Select the file and click Install Now.
6. Once your theme is successfully installed, you’ll see a activate link. Click on the Activate link, and you’ve successfully installed and activated CleanBlogg Pro. 

or

1. Upload the **cleanblog-pro** folder to the **/wp-content/themes/** directory
2. Activate "CleanBlog Pro" theme by going to **Appearance -> Themes**
